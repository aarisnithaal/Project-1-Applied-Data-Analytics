# 📊 Mobile Usage EDA Project

## 📌 Project Overview
This project analyzes mobile usage behavior using exploratory data analysis techniques.

## 🎯 Objectives
- Analyze app usage patterns
- Compare different user categories
- Identify relationships between variables

## 📈 Key Insights
- Most users spend around 3–4 hours daily on apps
- App usage and data usage show a positive relationship
- Higher behavior classes indicate heavier usage patterns

## 🛠 Tools Used
- Microsoft Excel
- Descriptive Statistics

- eda_report.pdf → Full analysis
- dataset.xlsx → Dataset used
